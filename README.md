# frontend_backend_basicProject

## Backend Setup

1. First set up the basic environment.
2. Run the command: npm init to initialize the project.
3. Install the necessary packages: npm install express mongoose.
4. Import Express and Mongoose in your project.
5. Use middleware for handling requests and responses.
6. Connect Mongoose to your database (e.g., mongodb://localhost/databasic).
7. Create the Schema and Model for your data.
8. Set up basic routes for handling API requests.


## Frontend Setup

1.First, set up the basic frontend environment.
2.Run the command: npm create vite@latest
3. Install the required packages: npm install tailwindcss axios
4. Import axios, useState, and useEffect in your React component.
5. Create a form with the required input fields (e.g., Reg Number, Name, Course, Department).
6.Store all input field values using useState and handle changes with onChange listeners.
7. Check whether all fields are filled (validate for empty inputs).
8. If inputs are not empty, process the data (submit it to the backend).
9.Save the submitted data into the database.
10.Display the stored data below the form in a dashboard or table view.
11. Include two buttons — Edit and Delete — for each record.
12. Finally, build and test the application.
